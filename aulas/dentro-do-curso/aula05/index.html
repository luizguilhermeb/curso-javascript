Como comentar em JS

Pode-se usar 2 formas:

  1. (comenta apenas 1 linha): // Exemplo 1
  2. (pode comentar várias linhas): /* Exemplo 2 */

Termos importantes para variáveis

Quando colocamos um valor dentro de uma variável: var n1 = 5 dizemos que var n1 "recebe" 5. Chamamos tal relação de "atribuição"

var também pode ser substituído por let

Há 3 formas de delimitar uma string dentro do Javascript:

  • var s1 = "Aspas duplas"
  • var s1 = 'Aspas Simples'
  • var s1 = `Crase`

É possível mudar o valor de uma variavel apenas fazendo uma nova atribuição

Nomear Variáveis

  • Podem começar com letra, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Coisas Importantes

Diferença de var e let

Var é funcional em toda a função, já let em apenas um bloco específico, estudar mais posteriormente.

Diferença de string com aspas duplas, aspas simples e crase no uso de strings javascript.

Basicamente aspas simples e duplas são intercabíveis, a mais diferente é a crase pois adiciona algumas funcionalidades ao uso da string.

Tipos Primitivos da Linguagem

Para saber o tipo primitivo de uma variável basta usar o comando typeof variável-qualquer

Principais

  • String
  • Number
  • Boolean

Demais

  • Infinity
  • NaN
  • Null
  • Undefined
  • Object
  • Array
  • Function
