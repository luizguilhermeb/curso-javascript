Condições

Condição Simples

Uma condição com apenas um if


if (vel > 60) { //condição simples
    console.log(`Você ultrapassou a velocidade permitida! MULTADO`);
}

Condição Composta

Uma condição com if e else


if (país == 'Brasil') {
    console.log(`Você é Brasileiro`)
} else {
    console.log(`Você Estrangeiro`)
}

Adições Importantes

Tanto em ids do HTML como variáveis do JS não é recomendável usar acentos por mais que seja possivel.
