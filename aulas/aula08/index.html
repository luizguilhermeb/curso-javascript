Operadores em Javascript (Parte 2)

Relacionais

  • > maior
  • < menor
  • >= maior ou igual
  • <= menor ou igual
  • == igualdade (não se restringe a Number ou String)
  • != diferente
  • === igualdade restrita (Se restringe a Tipos Primitivos)
  • !== Desigualdade restrita (Se restringe a Tipos Primitivos)

EX: 2 + 2 > 2 - 1 < 2 + 3 //SAÍDA: true

Lógicos

  • ! (NÃO) negação (se for true, afirma que é false, e vice-versa)
  • && (E) conjunção (precisa de dois resultados true)
  • || (OU) disjunção (precisa de apenas 1 resultado true)

Ordem de precedência: !, &&, ||

Ternário

teste ? true : falseSe teste for true mostra algo e vice-versa

EX: 5 > 2 ? "5 É MAIOR QUE 2" : "5 NÃO É MAIOR QUE 2" //SAÍDA: 5 É MAIOR QUE 2
