Eventos DOM

É qualquer mudança de comportamento do usuário em relação aos elementos HTML. EX: mouseenter, mousemove, mousedown, mouseup, click, mouseout. Basicamente há 2 formas de adicionar um event listener: 


<div id="area" onclick="clicar()">
  <p>Interaja</p>
</div>

e:


var area = document.querySelector('#area')
area.addEventListener('click', clicar)

Função

Na DIV exemplo foram aplicadas 3 funções, clicar(), entrar(), sair(). Cada uma delas é ativada a partir de um evento. O exemplo abaixo mostra a function clicar(): 


function clicar() {
  area.innerText = 'Clicou!'
  area.style.backgroundColor = 'blue'
}
